# MAYAJAGA App – Mini Project

## ภาพรวมโครงงาน (Overview)
MAYAJAGA เป็นโครงงานขนาดเล็กที่มุ่งเน้นการวิเคราะห์แนวคิดแอปพลิเคชันและการออกแบบระบบ
สำหรับแพลตฟอร์มสั่งซื้อเครื่องดื่ม โดยมีเป้าหมายเพื่อเชื่อมโยงผู้ใช้งาน ร้านค้า และผู้ดูแลระบบ
เข้าด้วยกันอย่างมีประสิทธิภาพ

โครงงานนี้จัดทำขึ้นเพื่อแสดงทักษะด้านการวิเคราะห์ความต้องการของผู้ใช้งาน
การออกแบบโครงสร้างระบบ และการสื่อสารแนวคิดของแอปพลิเคชันผ่าน Prototype

## วัตถุประสงค์ (Objectives)
- วิเคราะห์กลุ่มผู้ใช้งานตามอายุ อาชีพ และไลฟ์สไตล์
- กำหนด Functional Requirements ของระบบ
- ออกแบบ User Flow และโครงสร้างการทำงานของระบบ
- สร้าง Prototype เพื่ออธิบายแนวคิดและการทำงานของแอปพลิเคชัน

## บทบาทและหน้าที่ (My Role)
- Business Analyst / System Analyst  
- วิเคราะห์ความต้องการของผู้ใช้งาน (User Requirement Analysis)
- กำหนด Functional Requirements ของระบบ
- ออกแบบแนวคิดแอปพลิเคชันและหน้าจอการใช้งาน (Prototype) ด้วย Canva

## ฟังก์ชันหลักของระบบ (Key Features)
- ระบบสมัครสมาชิกและเข้าสู่ระบบ
- การค้นหาและสั่งซื้อเครื่องดื่ม
- การชำระเงินผ่าน QR / พร้อมเพย์
- ระบบจัดการคำสั่งซื้อสำหรับร้านค้า
- ระบบผู้ดูแล (Admin) สำหรับอนุมัติร้านค้า ตรวจสอบรีวิว และดูรายงานยอดขาย

## เครื่องมือที่ใช้ (Tools)
- Canva – ออกแบบ Prototype และหน้าจอแอปพลิเคชัน  
- SequenceDiagram.org – ออกแบบ Flow และโครงสร้างการทำงานของระบบ  

## ผลลัพธ์ที่ได้รับ (Outcome)
โครงงานนี้ช่วยพัฒนาทักษะด้านการคิดเชิงระบบ การวิเคราะห์ความต้องการของผู้ใช้งาน
และการออกแบบแนวคิดแอปพลิเคชัน รวมถึงการสื่อสารแนวคิดทางเทคนิค
ให้อยู่ในรูปแบบที่เข้าใจง่ายและนำไปใช้ต่อได้

> โครงงานนี้จัดทำขึ้นในรูปแบบ Mini Project ทางการศึกษา
> เพื่อแสดงทักษะด้าน System Analysis และ Application Design
